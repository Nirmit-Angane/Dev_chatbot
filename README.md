# 🤖 Dev Chatbot

A lightweight **Developer Chatbot built with Python** that runs as a **desktop overlay assistant**.  
It allows developers to ask questions, browse the web, and receive answers without switching applications.

The chatbot stays accessible while coding, browsing, or using other software, improving productivity and workflow.

---

## ✨ Features

- 🖥 Desktop overlay interface
- 🤖 AI powered chatbot
- 🌐 Web browsing capability
- 💾 Chat history storage
- ⚡ Fast and lightweight
- 🐍 Built with Python

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/NIRMIT-ANGANE/DEV_CHATBOT.git
```

### Enter Project

```bash
cd DEV_CHATBOT
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run

```bash
python src/main.py
```

---

## 🧠 How It Works

1. The program launches a **floating desktop chatbot**.
2. Users can ask questions.
3. The chatbot processes the query.
4. It fetches information from the web and returns answers.
